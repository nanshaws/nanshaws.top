## 🌟 核心模块：三大引力场

### 1. 📂 资源博学馆 (The Digital Archive)

这里是网站的灵魂。基于 WordPress 深度定制，沉淀了 nanshaws 多年来的技术探索：

- **深度技术专栏**：涵盖从 Nginx 调优到 WordPress 架构设计的实战干货。
- **AI 进化指南**：收录最新的大模型提示词（Prompts）技巧、虚拟卡订阅教程及出海支付方案。
- **精品资源分享**：不仅仅是软件，更是筛选后的“避坑指南”与效率模板。

### 2. 🛠️ 极客工具坊 (The Workshop)

拒绝平庸的网页工具，我们追求极致的交互：

- **开发者工具集**：代码格式化、解析器、以及针对 SEO 优化的自动化小工具。
- **生产力加速器**：一系列旨在缩短重复劳动时间的自研轻量化插件。

### 3. 🍳 旗舰产品：AI 厨房助手 (Project: Chef AI)

这是我们最新的明星项目！

- **技术架构**：采用 **Vue 3 + Tailwind CSS + PHP 8.1** 驱动。
- **核心逻辑**：对接阿里通义千问大模型，通过私有化后端保障 API 安全。
- **用户价值**：普通人只需输入食材，秒出专业级食谱，让 AI 真正走进烟火气。

### 4. 每日加密货币查询价格功能

这是我们的明星项目！

- **用户价值**：普通人只需输入币种。

## 🛠️ 技术栈 (Tech Stack)

为了保证网站在 2026 年依然具备顶级的响应速度与 SEO 表现：

- **前端渲染**：[Vue.js 3.x](https://vuejs.org/) (Composition API) - 极速响应，状态驱动。
- **样式体系**：[Tailwind CSS](https://tailwindcss.com/) - 现代、简洁、全平台自适应。
- **后端引擎**：[PHP 8.1+](https://www.php.net/) & [Nginx](https://www.nginx.com/) - 稳健的数据处理与安全转发。
- **内核驱动**：[Alibaba DashScope](https://help.aliyun.com/zh/dashscope/) - 提供强大的神经网络语义解析。
